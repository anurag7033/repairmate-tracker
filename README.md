# Anurag Mobile Repairing Centre

Mobile repair tracking and shop management web app for Anurag Mobile Repairing Centre (tracking.anuragmobile.in).

## Local development

Requires Node.js 18+ and npm.

```sh
npm install
npm run dev
```

The dev server starts on http://localhost:8080.

## Build

```sh
npm run build
```

## Tech

React, TypeScript, Vite, Tailwind CSS, shadcn/ui, Supabase (Postgres, Auth, Edge Functions, Storage), Razorpay.
